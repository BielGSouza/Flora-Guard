# 🌳 Flora Guard

## Sobre o Projeto

O **Flora Guard** é uma solução tecnológica voltada para a prevenção de incêndios florestais por meio da integração entre monitoramento espacial, análise inteligente de dados e dispositivos IoT.

O sistema utiliza informações coletadas por satélites para identificar condições ambientais favoráveis ao surgimento de incêndios, como altas temperaturas, baixa umidade e vegetação ressecada. Com base nesses dados, a plataforma calcula o nível de risco da região monitorada, emite alertas preventivos para autoridades competentes e pode acionar sistemas automatizados de microaspersão para auxiliar na mitigação dos riscos.

O objetivo é contribuir para a preservação dos biomas, proteção da biodiversidade e redução dos impactos ambientais causados pelas queimadas.

---

## Problema

Os incêndios florestais representam uma ameaça crescente para o meio ambiente, causando:

* Destruição da fauna e flora;
* Perda da biodiversidade;
* Emissão de gases de efeito estufa;
* Degradação dos ecossistemas;
* Impactos econômicos e sociais para comunidades locais.

Diante desse cenário, torna-se fundamental o desenvolvimento de soluções preventivas capazes de identificar riscos antes que os incêndios ocorram.

---

## Solução Proposta

O Flora Guard atua em três camadas principais:

### 🛰️ Monitoramento Espacial

Satélites coletam dados ambientais em tempo real, monitorando fatores como:

* Temperatura da superfície;
* Índices de vegetação;
* Umidade;
* Anomalias térmicas.

### 💻 Inteligência e Análise

A plataforma web processa os dados recebidos e calcula o risco de incêndio da região monitorada.

### 💧 Mitigação Preventiva

Quando um risco elevado é identificado, o sistema:

* Emite alertas automáticos para órgãos responsáveis;
* Aciona dispositivos IoT;
* Ativa sistemas de microaspersão em áreas estratégicas para aumentar a umidade da vegetação.
